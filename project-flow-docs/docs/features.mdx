# Features

This document outlines the key features of the AI Project Flow Generator application.

---

## Core Features

### 1. AI-Powered Generation
- Converts natural language descriptions into flowcharts using Google Gemini AI
- Automatically identifies pages, relationships, and hierarchy
- Generates optimized node positions

### 2. Interactive Flowchart
- **Drag & Drop**: Move nodes anywhere on canvas
- **Zoom Controls**: Zoom in/out with buttons or scroll wheel
- **Pan Navigation**: Click and drag to move viewport
- **MiniMap**: Overview of entire flowchart
- **Background Grid**: Visual alignment reference

### 3. Smart Positioning
- X-coordinates: 0, 250, 500, 750 (4 columns)
- Y-coordinates: 0, 100, 200, 300 (4 rows)
- Left-to-right logical flow
- Top-to-bottom hierarchy

### 4. Real-time Processing
- Instant generation with loading indicators
- Automatic viewport fitting
- Smooth transitions

## User Interface Features

| Feature | Description |

| **Error Messages** | Clear, actionable error descriptions |
