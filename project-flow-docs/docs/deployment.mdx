# Deployment Guide

This guide covers deploying the AI Project Flow Generator to various platforms.

---

## Prerequisites

- [ ] Completed application build
- [ ] Valid Gemini API key
- [ ] Production environment configured
- [ ] Tested locally with `npm run preview`

---

## Build for Production

```bash
# Install dependencies
npm install

# Create production build
npm run build

# Preview build locally
npm run preview
