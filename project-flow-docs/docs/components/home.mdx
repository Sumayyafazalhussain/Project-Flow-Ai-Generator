Component Overview
The Home component serves as the primary entry point where users interact with the AI to generate flowcharts from natural language descriptions.

State Management
State	Type	Purpose
input	string	Stores user's text description
nodes	array	Stores flowchart nodes from AI
edges	array	Stores flowchart connections
loading	boolean	Tracks generation status
error	string	Displays error messages
Core Function
handleGenerate()
javascript
- Validates user input (non-empty check)
- Sets loading state and clears previous errors
- Calls generateFlow() API function
- Updates nodes/edges on success
- Displays appropriate error messages on failure
- Resets loading state
