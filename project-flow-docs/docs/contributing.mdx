# Contributing Guidelines

Thank you for considering contributing to the AI Project Flow Generator!

---

## Code of Conduct

### Be Respectful
- Use inclusive language
- Respect different viewpoints
- Accept constructive criticism
- Show empathy to others

### Unacceptable Behavior
- Harassment or insults
- Publishing others' private information
- Unprofessional conduct

---

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- Git
- Google Gemini API key

### Fork & Clone

```bash
# Fork the repository on GitHub first, then:
git clone https://github.com/YOUR-USERNAME/Project-Flow-Ai-Generator.git
cd Project-Flow-Ai-Generator

# Add upstream remote
git remote add upstream https://github.com/Sumayyafazalhussain/Project-Flow-Ai-Generator.git
