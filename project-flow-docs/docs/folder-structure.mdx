# Folder Structure

This document provides a comprehensive overview of the AI Project Flow Generator's folder structure and file organization.

---

## Root Directory Structure
project-flow/
├── node_modules/ # Dependencies
├── public/ # Static assets
│ ├── vite.svg # Vite logo
│ └── tailwind-config.js # Tailwind configuration
├── src/ # Source code
│ ├── api/ # API integration
│ │ └── gemini.js # Gemini AI integration
│ ├── assets/ # Images, fonts, etc.
│ ├── components/ # Reusable components
│ │ └── FlowChart.jsx # React Flow wrapper
│ ├── pages/ # Page components
│ │ └── Home.jsx # Main application page
│ ├── App.css # App-specific styles
│ ├── App.jsx # Root component
│ ├── index.css # Global styles
│ └── main.jsx # Entry point
├── .env # Environment variables
├── .gitignore # Git ignore rules
├── eslint.config.js # ESLint configuration
├── index.html # HTML template
├── package-lock.json # Locked dependencies
├── package.json # Project metadata
├── post-config.js # PostCSS configuration
├── README.md # Project README
└── vite.config.js # Vite configuration

## Detailed File Breakdown

### Configuration Files

| File | Purpose |
|------|---------|
| `vite.config.js` | Vite build tool configuration |
| `tailwind.config.js` | Tailwind CSS framework configuration |
| `post-config.js` | PostCSS processor configuration |
| `eslint.config.js` | ESLint code quality rules |
| `.env` | Environment variables (API keys) |
| `package.json` | Project dependencies and scripts |

---

### Source Code Structure

#### API Layer (`src/api/`)
| File | Description |
|------|-------------|
| `gemini.js` | Handles Google Gemini AI integration, prompt engineering, and response parsing |

#### Components (`src/components/`)
| File | Description |
|------|-------------|
| `FlowChart.jsx` | Wraps ReactFlow library with controls, background, and minimap configuration |

#### Pages (`src/pages/`)
| File | Description |
|------|-------------|
| `Home.jsx` | Main application interface with state management, form handling, and API integration |

#### Entry Points
| File | Purpose |
|------|---------|
| `main.jsx` | React DOM rendering root |
| `App.jsx` | Root component wrapper |
| `index.html` | HTML template with root div |

#### Styles
| File | Purpose |
|------|---------|
| `index.css` | Tailwind CSS directives import |
| `App.css` | App-specific styles (optional) |

---

## Component Details

### FlowChart.jsx
- Wraps ReactFlow library
- Configures controls, background, and minimap
- Accepts nodes and edges as props
- Provides responsive container with fixed height

### Home.jsx
- Manages state (input, nodes, edges, loading, error)
- Handles form input and validation
- Calls `generateFlow()` API function
- Displays error messages and loading states
- Renders example prompts section
- Implements gradient background styling
